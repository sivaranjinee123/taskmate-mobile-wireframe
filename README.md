# TaskMate Mobile App Wireframing

## Project Overview
TaskMate is a mobile task management application designed to help users organize daily tasks, track deadlines, and improve productivity.

## Objective
Create low-fidelity wireframes for a mobile application using Figma.

## Screens Designed
1. Splash Screen
2. Login Screen
3. Dashboard
4. Add Task Screen
5. Profile Screen

## Design Process
- User Research
- User Persona Creation
- User Flow Planning
- Low-Fidelity Wireframing

## Tools Used
- Figma
